## 💎 О проекте

**Viht VPN Cyber Terminal** — это ультрасовременный веб-интерфейс, спроектированный для удобства настройки премиального VPN через клиент Happ и официального Telegram-бота. Проект использует концепцию **Glassmorphism**, продвинутые интерфейсные анимации и максимальную оптимизацию производительности.

## 🚀 Ключевые возможности

*   **🌀 Терминальный интерфейс**: Горизонтальные "Pill" переключатели, стеклянные карточки и неоновое свечение.
*   **⚡ 60 FPS Анимации**: Аппаратное ускорение (GPU), отсутствие блокировок основного потока.
*   **🤖 AI Preloader**: Уникальная стилизованная сцена загрузки страницы, которая придает проекту масштабность.
*   **📱 Mobile-First**: Адаптировано под смартфоны со строгим соблюдением Touch-зон и компактными меню.
*   **🔗 Telegram-Ready**: Бесшовная интеграция с ботом @VihtVPNbot в один клик.

## 🛠 Стек технологий

- **React 18** + **Vite** (Ультра-скоростная сборка)
- **Tailwind CSS 3** (Стилизация, градиенты, Glassmorphism)
- **Framer Motion** (Декларативные плавные State-анимации)
- **Lucide React** (Интерфейсная графика и иконки)

## 📐 Деплой на Vercel (За 1 минуту)

Проект полностью настроен и готов к моментальному развертыванию (Zero Config):

1. Загрузите этот код в репозиторий на ваш **GitHub**.
2. Перейдите на **[Vercel.com](https://vercel.com/)** и нажмите `Add New -> Project`.
3. Импортируйте этот репозиторий из списка ваших проектов GitHub.
4. Нажмите **Deploy** (Vercel сам определит, что это Vite-проект).
5. Сайт соберется и станет доступен по публичной ссылке через 30-40 секунд!
